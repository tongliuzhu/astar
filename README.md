## Getting Started

### Prerequisites
```sh
Ubuntu 16.04
OpenCV
```

### Downloading Source code
```sh
git clone https://github.com/tongliuzhu/serial-pid-controller.git
git checkout -b your_feature_branch origin/master
```

### Compile and Run Source code
```sh
cd astar && cd build
cmake .. && make
./astar
```
